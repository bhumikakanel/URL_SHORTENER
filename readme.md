#URL SHORTENER SAAS

A production-ready URL shortening service built with Next.js, Express.js, Redis (Upstash), Vercel, and Render.

##Features
1. Generate short URLs from long URLs
2. Redirect users to the original URL
3. Retrieve original URL using short URL ID
4. Click analytics tracking
5. URL validation
6. Copy short URL to clipboard
7. Cloud deployment with Vercel and Render
8. Redis-powered storage using Upstash

##Tech Stack

Frontend :
1. Next.js
2. React
3. Typescript
4. Tailwind CSS

Backend :
1. Node.js
2. Express.js

Databse : Redis (Upstash)


##Steps to run locally
1. Clone repository
2. Frontend :
   cd client
   npm install
   npm run dev
3. Backend :
   cd server
   npm install
   npm run dev
